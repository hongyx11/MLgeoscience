Lecture Notes created using [MkDocs](https://www.mkdocs.org).

## Set up environment
To get started install Mkdocs and Mkdocs material:
```
pip install mkdocs
pip install mkdocs-material
```

## Run locally
Type the following command in the terminal

```
mkdocs serve
```

## Build static files
Type the following command in the terminal

```
mkdocs build
```

## Publish on Github Pages

```
mkdocs gh-deploy
```
